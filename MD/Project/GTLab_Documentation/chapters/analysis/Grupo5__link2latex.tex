\section{Visão Geral}
The \texttt{link2latex.m} file is a MATLAB function designed to automate the generation of \LaTeX\ TikZ code for visualizing directed graphs[cite: 1]. It transforms numerical data—node positions and connectivity—into a format for professional vector graphics[cite: 1].

The code determines the target node indices $c$ by indexing the domain vector $d$ with the endomap $\phi$[cite: 1].
\begin{lstlisting}
c = d(phi);
\end{lstlisting}

This section iterates through nodes to generate TikZ strings, assigning unique identifiers and placing them at $(x, y)$ coordinates derived from the complex vector $g$[cite: 1].
\begin{lstlisting}
for i = 1:n
    nodes{i} = sprintf('\\node (n%d) at (%.2f, %.2f) {%d};', i, x(i), y(i), D(i));
end
